# A discord bot filled with different commands

## Commands

### Entertainment Commands

- `!ping` - Pong!
- `!dice` - Rolls a 6 sided dice
- `!flip` - A coin flip game (Heads or Tails)
- `!8ball` - A magic ball that answer with different responses to your questions!
- `!pick *yourchoices*` - Picks randomly from the choices you give

### Utility Commands

- `!serverinfo` - Shows server information
- `!userinfo *username*` - Shows user information of the mentioned user
- `!avatar *username*` - Shows the mentioned user's avatar
- `!roll *number choice*` - A dice with your given number of sides.

## Technologies Used

- Node.js
- Discord.js
- Javascript
- dotenv
